📖 使用说明

测试步骤：

  1. 将此 POC 页面部署到 Web 服务器
  2. 在手机上安装爱奇艺极速版
  3. 通过 deeplink 唤起应用并加载此页面
  4. 切换到 "JSBridge 测试" 标签页
  5. 点击 "检测 JSBridge" 按钮
  6. 查看检测结果

支持的 Deeplink 协议：

  • iqiyi:// - 主协议
  • iqiyilite:// - 极速版协议
  • https://lite.m.iqiyi.com/ - Universal Link

关键发现：

  • WebView 会注入 __$$_native_call_ 对象
  • 该对象有 invoke(String) 方法可被调用
  • 可以触发各种原生功能
