📱 ADB 命令 (最可靠)

如果浏览器方式不工作，请使用 ADB 命令：

# 通过 Universal Link adb shell am start -W -a android.intent.action.VIEW \ -d "https://lite.m.iqiyi.com/action/test?url=REPLACE_URL" \ com.qiyi.video.lite # 通过自定义协议 adb shell am start -W -a android.intent.action.VIEW \ -d "iqiyilite://mobile/dynamic_transfer?url=REPLACE_URL" \ com.qiyi.video.lite # 直接启动 Activity 并传参 adb shell am start -W -n com.qiyi.video.lite/.ui.activity.TransferFromOutActivity \ --es url "REPLACE_URL" \ --ez mCheckUrlSecurity false \ --ez mIsShouldAddJs true # 启动 CommonOnLineServiceActivity adb shell am start -W -n com.qiyi.video.lite/com.iqiyi.webcontainer.commonwebview.CommonOnLineServiceActivity \ --es url "REPLACE_URL" \ --ez mCheckUrlSecurity false \ --ez mIsShouldAddJs true
